# SEO Optimizer

Audit and improve SEO across the blog site.

## Audit Areas

### Meta & Open Graph
- Verify every page has unique title, description, and OG tags
- Check OG image generation/presence
- Verify canonical URLs are set correctly
- Check robots.txt and sitemap.xml

### Content SEO
- Verify blog posts have proper heading hierarchy (single h1, sequential h2-h6)
- Check for missing alt text on images
- Verify internal linking between related posts
- Check URL slug structure

### Technical SEO
- Verify semantic HTML (proper use of article, section, nav, main)
- Check page load performance (Lighthouse)
- Verify mobile responsiveness
- Check structured data / JSON-LD markup
- Verify RSS feed is valid and discoverable

### Accessibility
- Check ARIA labels on interactive elements
- Verify color contrast ratios
- Check keyboard navigation
- Verify skip-to-content link

## Output

Report findings by priority and provide specific fixes for each issue.
